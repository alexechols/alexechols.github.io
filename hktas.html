Hollow Knight TAS Assistant

Hollow Knight Tool Assisted Speedrun (TAS) Assistant is a suite of software tools, packaged into a standalone application focused on providing in-depth interfacing capabilities for the video game Hollow Knight. What started out as a simple tool to log in-game data in real time (i.e. positions, velocities) quickly turned into a group of computational tools, calculation tools, and information gathering tools based on need. The main GUI is written with Tkinter in Python, connecting to a Lua script which interprets data from a modified version of the game (C#).

Tool assisted speedrunning is a technique that takes playing a video game and turns it into an optimization problem. Tools such as frame-by-frame advance and savestates are employed to perform tasks in the game that would not be possible by a human player. Having detailed information about the current state of the game is vital to being able to do this efficiently, and thus there is need for software tools to help with this process. Standard practice places these vital pieces of information on the game screen, but due to the volume of information, it is often easier to move much of the information into a separate window. Hence, Hollow Knight TAS Assistant!

In addition to displaying information, a major component of this tool is its computational capabilites. Using in-game position and velocity data, standard data analysis techniques, and knowledge of numerical methods, I was able to derive analytical expressions for many of the in-game behaviors based on empirical data. This was then used to write a simulation tool, capable of recreating some of the in-game physics for algorithmically optimizing movements.
